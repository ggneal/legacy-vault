Step 1: Create GitHub Account

  1. Go to github.com
  2. Click "Sign up" (top right)
  3. Choose a username (this will be in your URL)
  4. Verify email and complete setup

Step 2: Create Repository

  1. Click the "+" icon (top right) ‚Üí "New repository"
  2. Repository name: legacy-vault
  3. Description: "Personal legacy information vault"
  4. Choose: Public (so anyone with link can access)
  5. Check: "Add a README file"
  6. Click "Create repository"

Step 3: Upload Your Vault File

  1. In your repository, click "Add file" ‚Üí "Upload files"
  2. Drag and drop your legacy-vault.html file
  3. Important: Rename it to index.html
  4. Scroll down, click "Commit changes"

Step 4: Enable GitHub Pages

  1. Click "Settings" tab (top menu)
  2. Scroll down left sidebar, click "Pages"
  3. Under "Source", select: Deploy from a branch
  4. Branch: main, Folder: / (root)
  5. Click "Save"
  6. Wait 1-2 minutes, refresh page
  7. You'll see: "Your site is live at https://username.github.io/legacy-vault/"

Step 5: Test & Share

  1. Open your URL in a private/incognito browser window
  2. Verify the password login works
  3. Copy the URL for your Emergency Access Card
  4. To update: Upload new index.html file (replaces old one)
